built different

Keyboard-native,
not keyboard-adjacent.

Every action is reachable without lifting your hands from the home row.

⎇
Live GitHub integration
PRs, issues, actions, and notifications update in real-time via background polling. No separate OAuth token — uses your existing gh auth session.
↺
Operation log + undo
Every git operation is tracked. Ctrl+Z undoes the last command. Ctrl+O opens the full operations timeline with timestamps.
≡
Stacked diffs, first-class
A complete Graphite-equivalent workflow — rebase all dependents, sync your stack, submit as a batch of PRs — entirely from the terminal.
◈
Hunk-level staging
Stage hunks, individual lines, or whole files with single keys. Toggle blame overlay with b to see commit info per line alongside your diff.
get started

Up and running
in four steps.

01
Clone and install
git clone https://github.com/nommyt/gubbi
cd gubbi && bun install
02
Build the binary
bun run build:cli
Symlinks gubbi to ~/.local/bin. Make sure it's in your $PATH.
03
Authenticate GitHub CLI
gh auth login
gubbi proxies all GitHub API calls through gh — no separate token needed.
04
Run inside any git repository
cd ~/your-project && gubbi
